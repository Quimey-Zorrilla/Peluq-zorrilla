PINK-SHERIFF: Es el sitio web de una peluqueria, que tiene como objetivo ser funcional y de facil interaccion para el cliente. En la cual se puedan consultar una
o varias cosas de acuerdo a como lo desee el usuario. Acompañado de contenido llamativo para el nicho de gente que accederia a este tipo de sitio, o a él cual 
esta orientado. 
A lo largo de la cursada fue variando de estilos en base a lo aprendido, llegando al objetivo de crear una página con estilo personal, saliendo de los 
estereotipos originales.
